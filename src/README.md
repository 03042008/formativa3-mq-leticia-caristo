🌌 Título do Projeto -GalaxyTours


 Esse projeto tem foco em interfaces modernas e responsividade, adapatado para vários dispositivos móveis. Ele é um site de vendas de ingressos para o espaço, ou seja, uma viagem para o mundo da lua; sendo uma das maiores agências de turismo espacial do mundo.
 Nesse projeto adicionamos elementos ao CSS,para deixa-lo responsivo, para que a esperiência do usuário seja a melhor possivel, trazendo mais conforto para nosso cliente. 
 
 

🚀 Tecnologias Utilizadas
HTML5 (Semântica)
CSS3 (Estilização e resposividade)

📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado:
 ├── index.html 
   └── src/
   ├── styles/ (Estilos da página) 
   ├── scripts/ (Lógica de interação)
   └── images/ (Ativos visuais)
    

🚀 Funcionalidades Implementadas

- Media Queries (Responsividade)
